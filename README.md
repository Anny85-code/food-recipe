## Getting Started

### Get a local copy

- git clone the project

```bash
git clone https://github.com/jaflih/recipe_app.git
```

```
cd recipe_app
```

### Install the app

```
bundle install
rails db:migrate
```

### Run the seeder

```
rails db:seed
```

### Run the app

- To run the app

```
rails s
```

```
Visit http://localhost:3000/ in your browser.
```

### Run the tests

- To run the tests

```
rspec spec/models
```
